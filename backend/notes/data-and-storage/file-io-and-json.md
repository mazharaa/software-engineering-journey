# File I/O and JSON

## What Problem Do File I/O Solve?

File I/O allows programs to read data from and write data to files data on disk. JSON is a structured text format for storing and exchanging data, it represents objects as key-value pairs, making it easy to store complex, nested data in a readable way.

## Tradeoffs

- No need DB
- Human-readable

## Examples

Save tasks to JSON file format after perform add/update/delete command.
